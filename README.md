# CivicSync

CivicSync is a modern, comprehensive GovTech platform designed to streamline the reporting, routing, and resolution of civic issues and complaints.

## 🚀 Key Features

*   **AI-Powered Priority Scoring & Auto-Routing**: Analyzes complaint descriptions using AI to automatically assign a priority level and route the issue directly to the relevant municipal department (e.g., Water Supply, Roads & Traffic, Electricity).
*   **Duplicate Complaint Detection (Deduplication)**: Automatically identifies similar existing complaints in the same geographic area to prevent redundant task creation and streamline tracking.
*   **Multilingual Voice Input**: Built-in voice recognition allows citizens to dictate their complaint titles and descriptions in multiple languages (English, Hindi, and Odia) for maximum accessibility.
*   **Automatic Geolocation Tracking**: Leverages browser location APIs and reverse-geocoding to pinpoint the exact location of civic issues without requiring manual input.
*   **Role-Based Portals**: Custom-tailored workflows and interfaces for all stakeholders:
    *   **Citizen Portal**: File new issues, attach photos, and track resolution timelines.
    *   **Worker Portal**: Manage on-ground issues, update progress statuses, and mark tasks as resolved.
    *   **Department Portal**: Manage division-specific responsibilities and assign local field workers.
    *   **Admin Dashboard**: Maintain a high-level overview, monitor analytics, and oversee entire city-wide operations.
*   **Responsive & Modern UI**: Built with a premium, accessible interface powered by Tailwind CSS to ensure a great user experience on both desktops and mobile devices.

## 🛠️ Tech Stack Foundational Technologies

*   **Frontend**: React (Vite), TypeScript
*   **Styling**: Tailwind CSS
*   **State Management**: Zustand
*   **Routing**: React Router DOM
*   **AI Inference**: Groq API
